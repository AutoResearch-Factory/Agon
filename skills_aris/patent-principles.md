# Patent Principles

Patent lifecycle principles — a parallel branch to the paper pipeline. Patent law standards (novelty, non-obviousness, written description, enablement) replace research standards.

## Patent Novelty

Research novelty is NOT patent novelty. Anticipation: single-reference test — does any single prior art disclose ALL claim elements? Non-obviousness: multi-reference test — would combination + common knowledge render the invention obvious to person of ordinary skill?

## Prior Art Search

Extract from invention: core inventive concept, technical problem, key technical features, relevant classification codes. Search patent databases + academic literature + product documentation. Rank by relevance: which claim elements covered, which not; highlight differentiating features.

## Patent Claims

Independent claims: broadest scope. Dependent: narrow with specific features. Structure: preamble + transitional phrase (comprising/including/consisting of) + body (elements defining invention). Must be clear and definite, supported by specification, novel, non-obvious. Multi-jurisdiction: CN (two-part form), US (single sentence, means-plus-function), EP (two-part, problem-solution).

## Patent Specification

The specification supports the claims — it is not a paper. Every claim element must be supported (written description requirement). Every reference numeral must appear in detailed description. Background sets up problem without admitting prior art as invention. Embodiments from broad to specific, alternatives included. Never use limiting language unless intentionally narrowing scope.

## Patent Embodiment

Broad to specific: general embodiment first, then variations. Each embodiment supports at least one claim element. Multiple embodiments show range without limiting scope. Reference numerals consistent across all embodiments. Invention decomposition: technical problem → core inventive concept → key technical features → advantages → alternatives.

## Patent Review

Examiner reviews for: claim definiteness, written description support, enablement, prior art differentiation, reference numeral consistency, jurisdiction compliance. Uses legal standards, not academic review criteria.

## Pipeline

invention structuring → prior art search → novelty check → claims drafting → specification → embodiment → figure description → patent review. Multi-jurisdiction awareness: CN/US/EP have different formatting and legal requirements.
