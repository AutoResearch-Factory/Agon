# Figure Description

Technical figure description with reference numerals — for patent drawings and formal technical diagrams.

## Numeral Assignment Rules

- Every component in every figure must have a reference numeral
- Every reference numeral must be explained in the specification
- Numeral series per figure: FIG.1 = 100-series (100-199), FIG.2 = 200-series (200-299)
- If the same component appears in multiple figures, use the SAME numeral
- Do NOT modify user-provided figures — only describe them
- Flag missing figures that claims require but user has not provided

## Drawing Description Format

One sentence each, consistent format. CN: "图1是[发明名称]的系统结构示意图". US/EP: "FIG. 1 is a block diagram illustrating the system architecture according to one embodiment."
