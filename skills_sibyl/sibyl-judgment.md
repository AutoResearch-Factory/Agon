# Research Judgment

## Backwards Reasoning (Revisionist)

When results contradict expectations: identify which assumption failed — which part of the mental model was wrong? Formulate a revised hypothesis consistent with new data. Re-examine old results under the new hypothesis. Design a targeted experiment that discriminates between old and new hypotheses.

## Engineering Realism (Pragmatist)

Prioritize: existing code to build on > simple method with rigorous implementation > complex method with sloppy implementation. Standard evaluation protocol, simplest publishable version first. Real-world constraints (latency, memory, throughput) matter — reviewers ask.

Feasibility: score across existing infrastructure, implementation simplicity, compute budget, community traction, reviewer expectations. True cost: GPU hours × seeds × ablation experiments. If cost exceeds budget, idea is infeasible. 20%-effort version demonstrating 80% of contribution.
